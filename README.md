# 🧟 Zombie Survival Game (C++)

Console-based zombie survival game built using Object-Oriented Programming concepts in C++.

## Features
- Player vs Zombies
- Ammo and Health System
- Score Tracking
- Random Zombie Movement
- Win & Game Over Conditions

## OOP Concepts Used
- Abstraction
- Inheritance
- Polymorphism
- Encapsulation

## How to Run

Compile:
g++ main.cpp -o game

Run:
./game
